# Frontend Assessment Project

This project is a responsive frontend application built using HTML, CSS, and JavaScript (ES6). The main goal of this assessment was to create a clean and modern e-commerce style landing page while following proper frontend development practices.

The project includes a responsive hero slider, product listing grid, and interactive tabs/accordion section. The UI is fully responsive and works across desktop, tablet, and mobile devices.

---

## Features

### Responsive Hero Slider
- Autoplay slider
- Infinite looping
- Navigation arrows
- Pagination dots
- Mobile swipe support
- CTA buttons

### Product Listing Grid
- 4 cards on desktop
- 2–3 cards on tablet
- 2 cards on mobile
- Product title
- Brand name
- Current price
- Original price

### Tabs / Accordion Section
- Interactive tabs for desktop view
- Accordion layout for mobile view
- Only one section visible at a time

### Accessibility
- Semantic HTML structure
- Proper button usage
- Accessible image alt text
- Keyboard-friendly interactions
- Clean UI hierarchy
